Check if class changes result in balanced class sizes.

This function checks whether changing students between classes within the academic unit (UC) would lead to a balanced distribution of students across those classes.

Time complexity: O(N) where N is the total number of classes in the UC

Parameters

destinyClassCodeThe class code of the destiny class, or an empty string if not changing to a specific class.
Returns
